# Edge strategy: vertical specialization over ecosystem gap-filling

The Demo Store positions the operator as a coffee/CPG-specialist Shopify freelancer rather than a generalist. Every visible feature should reinforce "this person understands coffee DTC inside-out" — subscription cadence, gifting, brew guides, wholesale enquiries — not "this person can build any store."

## Considered Options

- **Ecosystem gap-filling** — identify a feature missing from the Shopify App Store and build it into the Demo Store. Rejected: the operator has very little Shopify experience, and reliably spotting real gaps (vs. apps they haven't discovered) requires multi-year ecosystem exposure. High risk of building something that already exists or that nobody wants.
- **Execution edge** — same features as competitors but better polish, performance, and animation. Rejected as the *primary* edge: hard to advertise from a single demo, and any competent freelancer can claim it. Still pursued as a secondary quality bar.
- **Custom-code skill** — bespoke checkout extensions, custom Subscriptions logic. Rejected as the primary edge for the same low-experience reason — needs ecosystem fluency to know which custom code actually pays off.
- **Vertical specialization** — chosen. Defensible without ecosystem expertise (you can become a domain expert in coffee DTC much faster than an expert across all of Shopify), narrows the Client pool in a way that increases conversion within it, and gives the Demo Store a coherent identity instead of a generic "look at all the features" feel.

## Consequences

- Every Demo Store feature decision is judged against "does this make a coffee-DTC Client say 'they get my business'?" — generic-store features that don't reinforce vertical depth get cut.
- Marketing copy, case studies, and outreach must lean into coffee/CPG language. A generic "I build Shopify stores" pitch undermines the strategy.
- If the operator later wants to broaden to other verticals, this Demo Store becomes a sunk asset rather than a reusable one — a second specialist demo would be needed.
